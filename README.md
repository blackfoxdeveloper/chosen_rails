# ChosenRails

Chosen for rails

## Installation

Add this line to your application's Gemfile:

    gem 'chosen_rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install chosen_rails

## Usage

Add //= require 'chosen_rails' for application.js and *= require 'chosen_rails' for application.css
